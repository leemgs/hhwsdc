# 화성화옹지구 서부발전협의회 공식 웹사이트

**HHWSDC — Hwaseong Hwaong Western Development Council**

🌐 https://hhwsdc.github.io

## 소개

화성화옹지구 서부발전협의회(HHWSDC)의 공식 정책 홈페이지입니다.  
경기남부 국제공항(경기국제공항)의 화옹지구 입지 확정 및 조속한 추진을 위한 청원·정책 자료를 제공합니다.

## 주요 내용

- 화옹지구 경기국제공항 추진 당위성
- 경제적 효과 및 지역 주민 혜택
- 소음 우려 해소 전략
- 공식 청원서 및 정책 백서
